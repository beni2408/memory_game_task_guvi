Memory Game

Overview & Rules

The Memory Game is a simple yet engaging card-matching game built using HTML, CSS, and JavaScript (DOM manipulation). The game begins with a shuffled set of cards laid face down. Players click on two cards at a time to flip them and reveal their front faces. If the two cards match, they stay face-up; if not, they flip back after a short delay. The goal is to find all matching pairs using memory and observation skills. The game includes a restart button to reset the board and reshuffle the cards. It is designed to be fully responsive, working seamlessly across both desktop and mobile devices.
